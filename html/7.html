我的学习路径

2025年9月 - 2025年10月

网页设计入门

开始学习高等数学、html、css等基础课程 掌握HTML 搭建网页结构， CSS 美化页面样式，JavaScript 实现简单交互，比如轮播图、按钮点击效果。同时，自学Python编程语言，掌握了基本语法和数据结构。

2025年10月 - 2025年11月

网页设计工具与实操

学习排版、色彩搭配、视觉层级的基础规则，比如字体大小区分标题和正文，用配色传递页面风格，避免元素杂乱无章。用 VS Code 这类编辑器写代码，借助 Chrome 开发者工具调试页面；先模仿简单网页（比如个人介绍页、产品展示页）制作，再尝试原创设计。

2025年11月 - 2025年12月

制作出自己的第一个网页作品并参赛获奖

选取非遗官网为主题在自己与团队的构建下，完成一份功能比较完善的网页，我们在网页中实践了自学内容和老师课上所教内容如响应式布局、css网页优化、用javaScript实现简单的交互效果

2025年12月 - 未来

深度学习探索

摆脱万能布局，熟练使用 header nav main article section footer 等语义标签，明确页面结构；掌握 figure figcaption （图文组合）、time （日期时间）等专用标签，提升页面可读性和 SEO 效果。除了基础的 input button，还要掌握 select datalist textarea 的高级用法；学会用 label 绑定表单控件，用 required pattern 等属性做前端基础验证；了解 fieldset legend 对表单分组的作用，优化表单体验。学习设置 alt 属性（图片替代文本）、aria-* 系列属性，提升网页对辅助设备的支持；了解 HTML5 新特性在旧浏览器的兼容方案，比如通过降级处理适配低版本浏览器。
